# 🎬 Movie Recommender — Full Stack

A content-based movie recommendation system built with:
- **Notebook** — data processing + TF-IDF model + pickle export
- **Backend** — FastAPI + TMDB API + local TF-IDF recommendations
- **Frontend** — Streamlit UI with search, home feed, and details page

---

## 🚀 Setup Guide

### Step 1 — Get a TMDB API Key (free)
1. Go to https://www.themoviedb.org/settings/api
2. Create an account and request an API key
3. Copy the key

### Step 2 — Run the Notebook (Google Colab recommended)
1. Upload `notebook/movie_recommendation.ipynb` to Google Colab
2. Upload your `movies_metadata.csv` to `/content/`
   - Download from: https://www.kaggle.com/datasets/rounakbanik/the-movies-dataset
3. Run all cells top to bottom
4. The last cell saves 4 pickle files to `/content/`:
   - `df.pkl`, `indices.pkl`, `tfidf_matrix.pkl`, `tfidf.pkl`
5. Download those 4 files and copy them into `backend/`

### Step 3 — Set up the Backend
```bash
cd backend

# Create .env file
cp .env.example .env
# Edit .env and paste your TMDB API key

# Install dependencies
pip install -r requirements.txt

# Start the server
uvicorn main:app --reload --port 8000
```

Test it's working: http://127.0.0.1:8000/health

### Step 4 — Run the Frontend
```bash
cd frontend

pip install -r requirements.txt

streamlit run app.py
```

Open: http://localhost:8501

## 🐛 Bug Fixes Applied

### IndexError: positional indexers are out-of-bounds
**Root cause:** Duplicate movie titles in the dataset made `indices[title]` return a
Series instead of a scalar, which crashed `tfidf_matrix[Series]`.

**Fix (notebook):**
```python
indices = pd.Series(df.index, index=df['title'])
indices = indices[~indices.index.duplicated(keep='first')]  # ← this line
```

**Fix (backend):**
```python
def build_title_to_idx_map(indices):
    for k, v in indices.items():
        nk = _norm_title(k)
        if nk not in title_to_idx:   # first occurrence only
            title_to_idx[nk] = int(v)
```

### "Group Sex" recommended for "Toy Story"
**Root cause:** TF-IDF on overview text alone can match on coincidental rare-word overlap.
Genre information had equal weight to random description words.

**Fixes:**
1. Genres repeated 3× in `final_features` to boost their TF-IDF weight
2. Re-ranking: `final_score = 0.60×similarity + 0.30×vote_average + 0.10×popularity`
   — filters out low-rated obscure films even if they matched textually

## ☁️ Deploying to Render (free tier)

### Backend
1. Push `backend/` to a GitHub repo
2. Add your 4 `.pkl` files (or use Git LFS for large files)
3. Create a new Render Web Service:
   - Build command: `pip install -r requirements.txt`
   - Start command: `uvicorn main:app --host 0.0.0.0 --port $PORT`
   - Add env var: `TMDB_API_KEY=your_key`

### Frontend
Update `API_BASE` in `frontend/app.py` to your Render URL:
```python
API_BASE = "https://your-app.onrender.com"
```
Then deploy to Streamlit Cloud or another Render service.
